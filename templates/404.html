API Hakkında

FreeMusicAPI, YouTube Music veritabanındaki müzik, sanatçı, albüm ve çalma listesi verilerine kolay bir şekilde erişmenizi sağlayan RESTful bir API'dir. Bu API ile aramaları yapmak, sanatçı detaylarını görüntülemek, şarkı sözlerine erişmek ve daha fazlası için basit HTTP istekleri kullanabilirsiniz.

Tüm API yanıtları JSON formatında döndürülür ve isteğe bağlı olarak farklı dil desteği içerir.

Temel Kullanım

Aşağıdaki örnekte, API'nin temel kullanımını göstermektedir: 

// Popüler müzik trendlerini almak
fetch('https://freemusicapi.up.railway.app/API/trends?lang=tr')
  .then(response => response.json())
  .then(data => console.log(data))
  .catch(error => console.error('Error:', error));

API Endpoints

GET

/API/home

YouTube Music ana sayfasındaki tavsiyeleri ve popüler içerikleri getirir.

Parametreler

Parametre Tip Zorunlu Açıklama
limit Integer Hayır Döndürülecek maksimum öğe sayısı (varsayılan: 25)
lang String Hayır Sonuçların dili (varsayılan: tr)

Örnek İstek

GET /API/home?limit=10&lang=tr
GET

/API/mood/categories

YouTube Music'teki mood ve ortam kategorilerini getirir.

Parametreler

Parametre Tip Zorunlu Açıklama
limit Integer Hayır Döndürülecek maksimum kategori sayısı (varsayılan: 100)
lang String Hayır Sonuçların dili (varsayılan: tr)

Örnek İstek

GET /API/mood/categories?limit=20&lang=tr
GET

/API/mood/playlists

Belirli bir mood kategorisindeki çalma listelerini getirir.

Parametreler

Parametre Tip Zorunlu Açıklama
params String Evet Kategori parametresi
lang String Hayır Sonuçların dili (varsayılan: tr)

Örnek İstek

GET /API/mood/playlists?params=VLPL4fGSI1pDJn6jXS_Tv_N9B8Z0HTRVJE0S&lang=tr
